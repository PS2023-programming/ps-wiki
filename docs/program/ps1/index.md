## 简介

本页面是 2023 级问题求解（一）的大项目首页。

在该项目中，你需要：

+ 利用 ANSI 转义序列实现一个类似 ncurses 的控制台 UI 框架 - Minitui。
+ 利用 Minitui 完成一个具备沙盒编辑、游戏存档功能的回合制战略游戏。

本项目的日程安排详见[此处](../../schedule/ps1/)。
